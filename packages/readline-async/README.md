# `@kidscannon/mysql-migrate-cli`

## Install

```
$ npm i @kidscannon/mysql-migrate-cli
```

## Usage

```
$ MYSQL_HOST=example.com \
  MYSQL_PASSWORD=**** \
  MYSQL_DATABASE=db \
  MYSQL_SSL='{ "rejectUnauthorized": true }' \
  mysql-migrate migrate
```
